Free Music

React and Node.js

Group Project with Caelan Seto and Vlad Epelboym

Fully Functional Website for Professional Musicians. Musicians on Tour often need Sheet Music because its been misplaced or damaged. This site allows them to access scores from anywhere in the world as well as be able to upload for friends.

We Used React and Node.js with Express, Cors, JWT and MySql and also hosted it on an ec2 Instance with a S3 Bucket for the images and music files.
