Free Music

React et Node.js

Projet de groupe avec Caelan Seto et Vlad Epelboym

Site Web entièrement fonctionnel pour les musiciens professionnels. Les musiciens en tournée ont souvent besoin de partitions parce qu'elles ont été égaré ou endommagé. Ce site leur permet d'accéder aux partitions de n'importe où dans le monde ainsi que de pouvoir les télécharger pour des amis.

Nous avons utilisé React et Node.js avec Express, Cors, JWT et MySql et l'avons également hébergé sur une instance ec2 avec un compartiment S3 pour les images et les fichiers musicaux.
